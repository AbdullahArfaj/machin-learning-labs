in This lab i focuses on feature engineering for a classification task to predict Order_Status. Since the dataset was already clean, the workflow involved avoiding data leakage (removing future knowledge like Delivery_Time), creating new time-based and price-based features, handling high-cardinality categories, and training a Random Forest classifier. The model achieved a baseline accuracy of 85.19%, though it struggled with minority classes like "Cancelled" and "In Transit."
