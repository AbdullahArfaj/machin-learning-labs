Sample Project: KNN Classifier Tuning Project GoalThe objective of this project was to predict a binary Target Class (0 or 1) based on a set of anonymous numerical features. The primary challenge was to determine the most accurate "K" value for the K-Nearest Neighbors algorithm. The ProcessScaling: Standardized all features so that large numerical ranges wouldn't bias the distance-based algorithm.Splitting: Separated data into 70% training and 30% testing.Testing K=1: Built an initial model which resulted in an accuracy of 75%.Elbow Method: Ran a loop to calculate error rates for $K$ values between 1 and 40. ResultsBy plotting the error rates, we identified that K=30 was the optimal balance between bias and variance.ModelAccuracyInitial ($K=1$)75%Final ($K=30$)84% How to RunInstall dependencies: pip install pandas seaborn scikit-learnOpen 02-K Nearest Neighbors Assignment.ipynb.Run all cells to view the visualizations and the final classification report.ConclusionUsing the Elbow Method to choose $K$ improved the model accuracy by 9%. This demonstrates that the default $K=1$ is rarely the best choice for real-world datasets.
